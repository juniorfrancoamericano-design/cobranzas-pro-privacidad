# Política de Privacidad de Cobranzas PRO

**Última actualización:** 20 de julio de 2026

## 1. Introducción

Cobranzas PRO respeta la privacidad de sus usuarios. Esta Política de Privacidad explica qué información recopilamos, cómo la utilizamos y cómo la protegemos.

## 2. Información que recopilamos

La aplicación puede recopilar:

- Nombre y apellido.
- Dirección de correo electrónico.
- Número de teléfono.
- Información de clientes registrada por el usuario.
- Ubicación GPS (cuando el usuario concede permiso).
- Información del dispositivo necesaria para el funcionamiento de la aplicación.

## 3. Uso de la información

La información recopilada se utiliza únicamente para:

- Administrar cuentas.
- Gestionar clientes y cobranzas.
- Mostrar recorridos de los cobradores.
- Mejorar la seguridad del sistema.
- Cumplir con las funciones contratadas por el usuario.

## 4. Compartición de datos

No vendemos información personal.

Los datos únicamente pueden compartirse con proveedores necesarios para prestar el servicio, como:

- Firebase (Google)
- Stripe
- Google Maps

## 5. Seguridad

Aplicamos medidas de seguridad para proteger la información almacenada contra accesos no autorizados.

## 6. Conservación de datos

Los datos permanecerán almacenados mientras la cuenta esté activa o mientras sea necesario para prestar el servicio.

## 7. Derechos del usuario

El usuario puede solicitar la actualización o eliminación de sus datos comunicándose con nosotros.

Podremos actualizar esta Política de Privacidad cuando sea necesario. La versión más reciente siempre estará publicada en esta página.
